# Tài liệu nghiệp vụ Hammer

## Mục đích

Thư mục này chứa tài liệu BA chuẩn hóa cho toàn workspace.

Lưu ý: Đây là thư mục tài liệu BA chính theo quy ước hiện tại.

## Thứ tự đọc đề xuất

1. `knowledge-base.md`: nguồn tri thức chính, tổng hợp toàn bộ bối cảnh nghiệp vụ.
2. `hammer_wallet.md`: tài liệu chi tiết cho module Hammer Wallet.
3. `thuat-ngu.md`: từ điển thuật ngữ thống nhất giữa BA/Dev/QC.
4. `hammer_businessmodel.md`: mô hình business và capability map.
5. `kpi.md`: từ điển KPI và chỉ số vận hành.
6. `open-issues.md`: giả định mở, câu hỏi chưa chốt và rủi ro.

## Quy tắc cập nhật

- Ưu tiên cập nhật `knowledge-base.md` trước.
- Khi thêm tài liệu mới, ghi link vào file này.
- Mọi thay đổi thuật ngữ phải cập nhật lại `thuat-ngu.md`.
- Nếu cần tham chiếu lịch sử, đọc thêm các ghi chú trong thư mục `Quyennote`.
